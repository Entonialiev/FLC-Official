## 🧠 Что делают другие протоколы

### A2A (Agent2Agent)
- **Назначение:** Передача задач между агентами.
- **Что умеет:** Обмен сообщениями, координация.
- **Чего не умеет:** Оценивать последствия действий агентов.

### MCP (Model Context Protocol)
- **Назначение:** Подключение ИИ к внешним данным и инструментам.
- **Что умеет:** Вызов API, доступ к базам данных.
- **Чего не умеет:** Обнаруживать аномалии и останавливать цепочки.

### ATTP (Agent Trust Transfer Protocol)
- **Назначение:** Передача доверия между агентами.
- **Что умеет:** Аутентификация, авторизация.
- **Чего не умеет:** Оценивать влияние действий на систему.

---

## 🚀 Почему FLC — это прорыв

| Проблема | Решение FLC |
|----------|-------------|
| Агенты перегружают API | Обнаружение аномалий времени → остановка |
| Агенты вносят неверные данные | Обнаружение ошибок → остановка |
| Агенты выполняют опасные команды | Контекстные правила → блокировка |
| Нет доказательств для Bug Bounty | PoC-сигналы с payload_id |
| Нет аудита действий | Сохранение сигналов в JSONL |
